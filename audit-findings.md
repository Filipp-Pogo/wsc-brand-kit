# WSC Brand Kit — Full Audit Findings
**Date:** March 2026 | **Auditor:** Manus

---

## CATEGORY 1 — MISSING CONTEXT FOR NON-TENNIS/GOLF USERS

These are the most critical gaps. A new hire, vendor, or partner with no sports background will encounter these terms with no explanation.

### 1.1 APL (Athletic Performance Lab) — Never Explained
- **Issue:** APL appears 12+ times across the kit (signage, tone matrix, Instagram hashtags, channel guidelines, Claude prompt) but is never defined in the Terminology section or Brand Architecture section.
- **Impact:** A new front desk hire or marketing vendor has no idea what APL is, what happens there, or how to talk about it.
- **Fix:** Add APL to the Terminology section with a definition. Add it to Brand Architecture as a WSC fitness and performance programming lane, not as a Tier 1 sub-brand.

### 1.2 "Two Divisions / One System" — Used but Not Explained
- **Issue:** This tagline appears as an approved phrase but nowhere does the kit explain what the two divisions ARE (Tennis + Golf) or what "One System" means (shared Tier 1 academy methodology).
- **Impact:** Someone could use this phrase correctly in a caption but not understand what they're saying.
- **Fix:** Add an explanatory note in the Taglines section and Terminology section.

### 1.3 "Full-Time Tennis" / "Full-Time Golf" — No Program Description
- **Issue:** These are listed as approved terms but there is no explanation of what Full-Time Tennis or Full-Time Golf actually means as a program structure.
- **Impact:** A new team member cannot explain the program to a prospective family.
- **Fix:** Add brief program descriptions to the Terminology or Brand Architecture section.

### 1.4 "S&C" — Unexplained Abbreviation
- **Issue:** "S&C / APL-Specific Marketing" appears in the Tone Matrix but S&C (Strength & Conditioning) is never defined.
- **Fix:** Spell it out in the Tone Matrix row.

### 1.5 "UTR / USTA / USGA / ITA" — Referenced Implicitly, Never Explained
- **Issue:** The kit mentions "college placement" and "alumni destinations" (Penn, Harvard, Stanford, etc.) but never explains the pathway — USTA rankings, UTR ratings, recruiting process — that makes these outcomes possible.
- **Impact:** A non-tennis person writing about the pathway has no framework for what makes Tier 1 different.
- **Fix:** Add a "Program Context" explainer to Brand Architecture or a new "What We Do" section.

### 1.6 "Uneekor Launch Monitor" — Unexplained Technology Reference
- **Issue:** Referenced in Core Values as a proof point but never explained.
- **Fix:** Add a brief parenthetical or glossary entry.

### 1.7 "Clawdbot" — Unexplained Reference
- **Issue:** Referenced in Core Values ("Clawdbot's brand readiness platform") but never explained anywhere in the kit.
- **Fix:** Either add a brief explanation or remove the reference if it's not relevant to brand communications.

## CATEGORY 2 — INCONSISTENCIES

### 2.1 "Tagline" Inconsistency — "Train Without Limits" vs. "Two Divisions / One System"
- **Issue:** "Train Without Limits" is listed as a WSC secondary tagline. But "Two Divisions / One System" is listed as a Tier 1 tagline. However, in the Claude System Prompt, "Train Without Limits" is attributed to Tier 1, not WSC.
- **Fix:** Correct the Claude System Prompt attribution.

### 2.2 Brand Architecture — APL Listed as a Brand, Not Explained in Context
- **Issue:** APL appears in the Brand Architecture relationship diagram data but is not shown in the visual diagram itself (only Tier 1 and WSC appear). A user reading the architecture section gets an incomplete picture.
- **Fix:** Add APL as its own WSC fitness and performance lane in the diagram.

### 2.3 Tone Matrix — "Athlete / Coach Communications" Has No WSC Equivalent
- **Issue:** The Tone Matrix has a Tier 1 row for "Athlete / Coach Communications" but no WSC equivalent for member/family communications. This creates an asymmetry that implies WSC doesn't communicate directly with members.
- **Fix:** Add a WSC "Member / Family Communications" row to the Tone Matrix.

### 2.4 Instagram Guidelines — Hashtag "#APL" Used Without Explanation
- **Issue:** "#APL" and "#AthleticPerformanceLab" appear in the secondary hashtag list but APL is never explained in the Instagram section.
- **Fix:** Add a brief note explaining APL in the hashtag context.

### 2.5 Voice Constants — No WSC-Specific Examples
- **Issue:** The Voice Constants section has DO/DON'T examples but most examples lean Tier 1 (training, performance, athletes). WSC-specific examples (family events, facility communications, community content) are underrepresented.
- **Fix:** Add at least one WSC-specific example per Voice Constant.

### 2.6 Anti-Patterns — "Sales Content in Training Environments" Example Uses Tennis Courts
- **Issue:** The example says "A 'Summer Special — 20% off enrollment' sign posted near the courts" — but the kit never establishes what "the courts" means to someone unfamiliar with the facility.
- **Fix:** Minor — add "tennis/golf courts" for clarity.

### 2.7 Quick Reference Card — Missing APL and Program Structure
- **Issue:** The Quick Reference Card (the print-ready card for non-desk staff) doesn't mention APL, Full-Time Tennis, or Full-Time Golf — the three things non-desk staff are most likely to be asked about.
- **Fix:** Add a "Programs" row to the Quick Reference Card.

### 2.8 Email Guidelines — No Tier 1 Transactional Email for Golf
- **Issue:** The enrollment confirmation and session reminder email templates reference tennis-specific language (courts, sessions) but there is no golf-specific variant or note about adapting for golf.
- **Fix:** Add a note about sport-specific adaptation in the email templates.

### 2.9 Co-Branding — No Guidance on Tournament Co-Branding
- **Issue:** The Co-Branding section covers sponsors, schools, and media partners but doesn't address tournament co-branding — which is likely the most common co-branding scenario for a tennis/golf academy.
- **Fix:** Add a "Tournament & Event Co-Branding" sub-section.

### 2.10 Brand Checklist — No "Sport Context" Check
- **Issue:** The Brand Checklist doesn't include a check for whether sport-specific language is appropriate (e.g., using tennis terminology in a golf communication).
- **Fix:** Add a "Sport Context" category to the checklist.

## CATEGORY 3 — CONTENT GAPS

### 3.1 No "What We Do" Plain-Language Explainer
- **Issue:** The kit assumes the reader already knows what Tier 1 Performance and WSC do. There is no plain-language paragraph that explains: "We run a tennis and golf performance academy for youth athletes inside a 67-acre sports club in Woodinville, WA."
- **Impact:** The highest-risk gap for non-sports users. Without this, the entire kit is built on an assumed foundation.
- **Fix:** Add a "Context" or "What We Do" panel to the Brand Architecture section.

### 3.2 No Photography/Videography Guidelines
- **Issue:** The kit tells you what to say but not what to shoot. No approved angles, lighting conditions, color grading references, or shot list.
- **Impact:** Every external photographer or videographer hired needs this before a shoot.

### 3.3 No Paid Advertising Rules
- **Issue:** No guidance on what changes when content is paid vs. organic — ad copy length, CTA rules, claim restrictions.

### 3.4 No Physical Address / Location Context
- **Issue:** "67-acre campus in Woodinville, WA" is mentioned but the full address, nearest landmarks, or location context is never provided. Relevant for signage, event communications, and partner materials.

### 3.5 Version Changelog — Only Shows Kit Versions, Not Guideline Changes
- **Issue:** The changelog tracks Brand Kit app versions but doesn't document changes to the actual brand guidelines themselves (e.g., "March 2026 — Added golf to Two Divisions tagline").
- **Fix:** Separate "Kit Updates" from "Guideline Updates" in the changelog.
